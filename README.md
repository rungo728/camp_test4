# コンセプト

構造としてはいたって単純な掲示板アプリではあるが、javascript を使って動的な動きのある見やすいサイトにする。
基本となる会員登録・ログイン・ログアウトや、投稿した記事の確認画面や、画像のアップロード、Twetter との連携ややったこのない機能にチャレンジする。
